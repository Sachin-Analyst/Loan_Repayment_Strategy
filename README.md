## Project Description

This project addresses a critical financial challenge: securing a ₹15,00,000 loan while identifying a repayment plan that fits a strict monthly budget. By analyzing three different institutions Left Join Bank, Y Axis Bank, and CBI Bank—this dashboard identifies the most efficient path to debt-free status.


### Key Features
-*Comparative Analysis*
-*Budget Filtering*
-*Interest Optimization*
-*Decision Support*
